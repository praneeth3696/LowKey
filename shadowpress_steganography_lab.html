// how it works
every pixel = 3 bytes (R, G, B). each byte's last bit (LSB) is flipped to carry your message. the visual difference is ≤1/255 per channel — invisible to the human eye. a 1080×720 image can hide ~97KB of data in its noise.

// the art of invisible communication
01
Image LSB
hide data in the least significant bit of every pixel's RGB channel. modification is ≤1/255 — imperceptible to humans.
02
Text Steganography
embed binary-encoded messages using invisible Unicode characters (zero-width space, ZWNJ) between normal text glyphs.
03
Audio LSB
modify the LSB of WAV audio samples. the human ear can't detect ±1 amplitude changes — perfect for covert audio payloads.
04
Steganalysis
detect hidden data by visualizing LSB planes, chi-square statistical tests, or histogram anomalies in pixel distributions.
05
Whitespace Coding
encode messages using patterns of spaces and tabs at line endings — invisible in rendered text, readable in raw bytes.
06
Metadata Injection
EXIF fields in images, ID3 tags in MP3s, and document properties in PDFs can all carry hidden payloads undetected.
// historical context
steganography predates computing by millennia. ancient greeks tattooed messages on shaved heads, waited for hair regrowth, then sent the messenger. WWII saw invisible ink, microdots, and null ciphers. today's digital steganography is used for watermarking, covert channels, and censorship evasion. the core principle hasn't changed: the best hidden message is one whose existence goes unsuspected.
